# SATVI Data Portfolio

Data enthusiast with experience in **R, Excel, and Dotmatics**. This portfolio highlights projects in **data management, validation, and visualization**. 

The included project demonstrates:
- **Data cleaning and standardization** of survey datasets  
- **Data quality checks**, including missing values, duplicates, and impossible values  
- **Exploratory data analysis and visualization** of numeric and categorical variables  
- **Generation of actionable insights** through structured outputs and plots

Tools and libraries used: **R**, `tidyverse` (`dplyr`, `tidyr`, `readr`, `ggplot2`).

All scripts save results and visualizations in the `outputs/` folder, making it easy to review dataset quality and explore the data.
